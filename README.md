Project Overview

This project explores the implementation of GQSP techniques for simulating quantum time evolution and polynomial transformations of unitary matrices.

The work includes:

construction of GQSP circuits,
angle optimization using COBYLA,
polynomial validation,
error analysis,
applications to Hamiltonian simulation,
exploration of molecular systems such as H3.
